# 🤖 AI Air Canvas

An AI-based hand gesture controlled drawing application built using Python, OpenCV and MediaPipe.

## ✨ Features

- Real-time hand tracking
- Finger gesture detection
- Air drawing using hand movement
- Color control using keyboard
- Clear and save drawing options

## 🛠️ Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

## 🚀 How to Run

1. Clone the repository

2. Install requirements:
pip install -r requirements.txt

## 🎯 Project Purpose

This project explores computer vision and human-computer interaction using AI-powered hand tracking.
